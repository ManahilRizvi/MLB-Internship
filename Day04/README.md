# Module 3

This repository contains the manual implementation of fundamental image processing operations using **Python**, **OpenCV**, **NumPy**, and **Matplotlib**.

The main objective of this task is to understand how common image operations work internally by implementing them **manually with nested loops**, instead of relying on OpenCV's built-in functions.

---

## 📌 Topics Covered

- Reading Images
- Saving Images
- Image Copy
- Cropping
- Resizing
- Rotation
- Flipping
- Translation
- Scaling
- Image Padding
- Image Normalization

## 🛠 Technologies Used

- Python 3
- OpenCV (`cv2`)
- NumPy
- Matplotlib
