# Day 02 - Image Fundamentals (Module 1)

This folder contains the practical implementations of **Module 1: Image Fundamentals** completed as part of the **MLB Internship**.

## 📚 Topics Covered

- Image Loading
- Image Dimensions
- Image Channels
- Pixel Manipulation
- Mean, Minimum and Maximum Pixel Intensity
- Pixel Counting
- RGB Channel Separation
- RGB to Grayscale Conversion
- Image Memory Calculation

---

## 📝 Tasks

### ✅ Task 1
Load an image and display:
- Height
- Width
- Number of Channels
- Data Type
- Total Number of Pixels

---

### ✅ Task 2
- Change the color/value of 5 specific pixels.
- Save the modified image.

---

### ✅ Task 3
Manually calculate:
- Mean Pixel Intensity
- Minimum Pixel Value
- Maximum Pixel Value

> **Note:** No built-in image processing functions such as `np.mean()` or `cv2.mean()` were used.

---

### ✅ Task 4
Count:
- Total Pixels
- Black Pixels
- White Pixels
- Pixels with Intensity greater than 200

---

### ✅ Task 5
Manually split an RGB image into:
- Red Channel
- Green Channel
- Blue Channel

Each channel is saved as a separate image.

---

### ✅ Task 6
Convert an RGB image to Grayscale manually without using `cv2.cvtColor()`.

---

### ✅ Task 7
Calculate image memory using:
- Height
- Width
- Channels
- Bit Depth

Memory is displayed in:
- Bytes
- Megabytes (MB)

---
